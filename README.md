# Frontend Mentor - Social links profile

A responsive social links profile card built with HTML and CSS, featuring a clean dark theme design.

## Overview

This project displays a personal profile card with social media links in a centered, card-based layout. The design is fully responsive and adapts to different screen sizes.

## Features

- **Responsive Design**: Adapts seamlessly from mobile to desktop screens
- **Dark Theme**: Modern dark color scheme with green accent colors
- **Clean Layout**: Centered card design with profile information and social links
- **Custom Styling**: Uses CSS custom properties for easy theme customization
- **Google Fonts**: Integrated Inter font family for modern typography

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Inter)

## Project Structure

```
.

    └── images/
```

## CSS Features

- **CSS Custom Properties**: Color scheme defined using CSS variables
- **Flexbox Layout**: For centering and alignment
- **Media Queries**: Responsive breakpoint at 768px
- **Modern Units**: Uses rem units for scalability

## Color Scheme

- Background Container: `#141414`
- Card Background: `#1f1f1f`
- Link Background: `#333333`
- Accent Color (Location): `#c5f82a`
- Text Color: White

## Responsive Breakpoints

- **Mobile**: Default (up to 767px)
- **Desktop**: 768px and above
  - Container width: 768px
  - Card width: 384px

## Setup

1. Clone or download the project files
2. Ensure all assets are in the correct folder structure
3. Open `index.html` in your web browser

## Customization

To customize the profile:

1. **Profile Information**: Edit the text content in `index.html`
2. **Profile Image**: Replace `avatar-jessica.jpeg` with your own image
3. **Social Links**: Update the `href` attributes in the anchor tags
4. **Colors**: Modify the CSS custom properties in `:root` selector in `style.css`

## Browser Support
